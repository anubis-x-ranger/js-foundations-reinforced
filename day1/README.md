# Notes for Node
Node.js is a server-side JavaScript runtime built on the V8 engine.Unlike browser JavaScript, Node provides access to operating-system capabilities through built-in modules such as filesystem (fs), HTTP, crypto, OS, process, and child_process.

Node uses an event-driven, non-blocking I/O model, making it efficient for scalable, I/O-heavy workloads.

It also allows integration with native code through the Node API (N-API), enabling high-performance extensions written in languages like C or C++.
