# JS Foundations Reinforced

A structured engineering project to strengthen core **JavaScript and Node.js fundamentals** through real coding exercises and practical mini-tools.

This repository is part of a focused effort to build strong backend and system-level JavaScript skills required for building scalable applications and SaaS systems.

---

## Project Goal

The goal of this project is to deeply reinforce:

* JavaScript fundamentals
* Node.js runtime concepts
* File system operations
* CLI tool development
* Modular code architecture
* Git discipline through daily commits

Instead of passive learning, this repository focuses on **building small but real utilities**.

---

## Tech Stack

* JavaScript (ES6+)
* Node.js
* Node File System (fs)
* Git & GitHub

## Day 1 Project

### File Statistics CLI

A simple Node.js CLI tool that reads a file and prints useful statistics.

Example output:

```
File: example.txt
Lines: 24
Words: 182
Characters: 1054
```

Concepts practiced:

* Node.js runtime
* CLI arguments
* File reading using `fs`
* String processing
* Modular JavaScript

---

## How to Run

Clone the repository:

```
git clone https://github.com/anubis-x-ranger/js-foundations-reinforced.git
```

Navigate into the project:

```
cd js-foundations-reinforced
```

Run the CLI tool:

```
node day1/file-stats-cli.js example.txt
```
